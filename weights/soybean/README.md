# Soybean Checkpoints

These trained soybean models are stored with Git LFS:

- `soybean_rgb_nerfacto_128.pth`: RGB-stage Nerfacto checkpoint.
- `soybean_singlepod_invnerf_128.pth`: single-pod InvNeRF-Seg fine-tuned checkpoint.
- `soybean_doublepod_invnerf_128.pth`: double-pod InvNeRF-Seg fine-tuned checkpoint.

Use `git lfs pull` after cloning if the checkpoint files are present only as LFS pointer files.
